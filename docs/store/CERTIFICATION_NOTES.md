Use this text in Partner Center "notes for certification" after updating package versions and test evidence.

## Suggested notes

SnipFocus is a local-first desktop capture and editing application. Core flows are:

- fullscreen recording
- window recording
- trim and style in the built-in editor
- local export

The app may request access to:

- screen capture
- microphone
- webcam
- user-selected files and export destinations

The app includes a support-bundle export action for private-flight diagnostics. No account is required for core flows.

For this submission we validated:

- clean install and launch
- fullscreen recording
- window recording
- local export
- x64 package
- arm64 package
- WACK on final Store packages
