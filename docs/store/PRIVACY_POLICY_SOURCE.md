# SnipFocus Privacy Policy

Publish this content at a public URL before Store submission, for example:

- `https://snipfocus.app/privacy`

## Summary

SnipFocus is a local-first Windows desktop app for screen capture, recording, trimming, styling, and exporting videos. By default, SnipFocus processes recordings on the device and does not upload captured media to a cloud service.

## Data we may access

- screen recordings and screenshots the user explicitly captures
- microphone audio when the user enables microphone recording
- camera video when the user enables webcam recording
- files the user explicitly opens or exports
- local focus analytics data if the optional focus features are enabled

## How data is used

- to preview, edit, trim, style, and export recordings locally
- to store user preferences and optional local analytics on the device
- to generate support bundles only when the user explicitly exports one

## What we do not do by default

- we do not upload recorded video or audio to a cloud service by default
- we do not require account creation for core recording or editing flows
- we do not sell personal data

## Local storage

SnipFocus stores temporary processing files in the app temp directory and preferences/support files in the app data directory. Exported recordings are saved only to locations the user selects or to user library folders used by the app flow.

## Support bundles

If the user exports a support bundle, it may contain runtime information, feature flags, recent capture health metrics, and recent local FFmpeg error logs. Users should review support bundles before sharing them.
