# SnipFocus Microsoft Store Listing Draft

## Short description

Capture your screen, trim the result, style it, and export polished demo videos locally.

## Full description

SnipFocus is a local-first Windows screen recording and editing app built for fast demo creation. Capture fullscreen or window recordings, trim the clip, add clean styling, and export without moving your media to the cloud.

Core flow:

- Record
- Trim
- Style
- Export

Highlights:

- fullscreen and window recording
- webcam overlay support
- local editing and export
- bundled export pipeline for packaged releases
- support bundle export for private-flight issue reports
