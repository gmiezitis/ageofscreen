# Store Metadata Links

Use these public links in Partner Center after the repository is public.

## Website URL

Use:

https://github.com/gmiezitis/ageofscreen

Do not use:

https://github.com/gmiezitis/ageofscreen.git

The `.git` URL is a clone endpoint, not a user-facing website page, and can fail Microsoft Store metadata validation.

## Support URL

https://github.com/gmiezitis/ageofscreen/issues

## Privacy Policy

Use:

https://gmiezitis.github.io/ageofscreen/privacy.html

This repository publishes `privacy.html` through the `Publish GitHub Pages` workflow. After making the repository public, run that workflow or push to `main`, then verify the URL opens in a private/incognito browser window before resubmitting to Partner Center.
