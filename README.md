# AgeofScreen

AgeofScreen is a local-first Windows screen capture and media editor for making clear product demos, tutorials, walkthroughs, and support clips.

The app is designed around a fast capture flow: open the launcher, record or snip the screen, then edit the result with trimming, crop tools, annotations, webcam overlay, cursor-aware focus effects, screenshots, image clips, text overlays, and export options.

## Highlights

- Capture the full screen, a region, or a selected window.
- Record screen videos with optional webcam and cursor/click context.
- Open recordings directly in the media editor.
- Trim clips, crop the frame, add images, text, arrows, drawings, and step markers.
- Build mixed timelines with video clips, screenshots, imported pictures, audio, and annotations.
- Export polished videos and edited screenshots with AgeofScreen branding.
- Runs primarily on-device. Core capture and editing workflows do not require an account.

## Windows Support

AgeofScreen targets modern Windows desktop devices.

- Windows 10 version `10.0.19041.0` or later.
- Windows 11.
- `x64` and `arm64` Store packages.

## Privacy

AgeofScreen is built as a local-first app. Screenshots, recordings, project files, and exports stay on the user's device unless the user chooses to share them.

The privacy policy source is included in this repository at [`privacy.html`](privacy.html).
